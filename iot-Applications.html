Activity 1

IoT apps analyse vast volumes of linked sensor data in the cloud using machine learning algorithms and help getting real time data as notification on dashboards. It is a network of physical objects - "things" - with embedded sensors, software and other technologies to connect and exchange data with other devices and systems on the Internet. The main features of IoT appliction are analyzing, security, accuracy and real time data etc. It is used in almost every sphere of life for example: Online shopping, healthcare, industrial use, smart homes, agricultural use and many more.

The control interface is a protocol that allows communication between peripheral devices and the microcontroller. Depending on the application, a special control interface may be required. It is a border that contains a collection of methods for enforcing security regulations and controlling the flow of information between networked information systems. It is very flexible and secure. Different devices use different interfaces for example- TVremote has a user interface that consists of various buttons, and devices such as Amazon Echocan be controlled with voice commands.
